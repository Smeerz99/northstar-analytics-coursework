# NorthStar Analytics Coursework

This repository contains the Google Colab notebooks, code, and outputs for the NorthStar Urban Mobility and Logistics coursework.

## Structure
- `data/raw/` = original dataset files
- `data/processed/` = cleaned datasets
- `notebooks/` = Colab notebooks
- `outputs/figures/` = charts
- `outputs/tables/` = exported tables
